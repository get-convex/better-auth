Here's a list of Better Auth plugins that currently 100% work with the component:

- [x] [Two Factor](https://www.better-auth.com/docs/plugins/2fa)
- [x] [Username](https://www.better-auth.com/docs/plugins/username)
- [x] [Magic Link](https://www.better-auth.com/docs/plugins/magic-link)
- [x] [Email OTP](https://www.better-auth.com/docs/plugins/email-otp)
- [x] [SSO & Email/Password](https://www.better-auth.com/docs/authentication/email-password)

A few other plugins partially work, please provide feedback on the [Discord](https://discord.gg/convex-dev) server if you have any issues.

This project aims to have 100% support for all Better Auth plugins at some point, to follow the development of this component, join the [Discord](https://discord.gg/convex-dev) or check out the [GitHub](https://github.com/get-convex/better-auth) repository.
